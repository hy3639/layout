  • 사용 여부는 버튼을 선택 시 교차로 출력 됩니다. 수정 후 [저장] 시 반영 됩니다.

  • 대분류 명 수정 : '대분류 선택박스'를 선택하여 수정 가능하며, 하단에 있는 '추가하기'를 선택하여 추가 가능합니다.
  • 소분류 명 수정 : 입력 영역에 출력된 명칭을 수정할 수 있으나, 업무 유형이 바뀌는 명칭으로의ㅣ 수정은 자제하시기 바랍니다.
    예시: 목돈 굴리기 => 예금(수정O), 목돈 굴리기 => 대출(수정X)
    수정 시 기존 명칭이 전체 바뀌게 되오니, 주의 하시기 바랍니다.
  • 글자수 제한: 대 분류 10자, 소 분류 10자
  • 업무 유형 삭제 불가 합니다.
  • 업무 유형 코드는 수정 불가 합니다.
